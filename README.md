# Table

A virtual table-top for basic roleplaying games.

## Requirements

- Ruby 2.6.4
- Yarn
- Postgres

## Setup

```
$ bin/setup
```

## Testing

Tests are handled by rspec:

```
$ rails spec
```

## Linting

Run linting before merging. Rubocop is setup:

```
$ rails lint      # Runs lint:ruby
$ rails lint:ruby # Runs rubocop
```
